# Trabalhos

Pasta destinada aos trabalhos e projetos da disciplina de Séries Temporais.

## Estrutura sugerida

Organize cada trabalho em uma subpasta com nome descritivo:

```
trabalhos/
├── trabalho-01/
│   ├── enunciado.pdf
│   ├── relatorio.pdf
│   └── analise.ipynb
├── trabalho-02/
│   └── ...
└── ...
```
