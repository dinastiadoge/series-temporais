# Scripts

Pasta destinada aos scripts Python utilitários e de apoio à disciplina de Séries Temporais.

## Estrutura sugerida

Organize os scripts por tema ou funcionalidade:

```
scripts/
├── exploracao/
│   └── analise_exploratoria.py
├── modelos/
│   ├── arima.py
│   └── sarima.py
├── visualizacao/
│   └── plots.py
└── utils.py
```

## Dependências comuns

- `pandas`
- `numpy`
- `matplotlib`
- `statsmodels`
- `scikit-learn`
