# series-temporais

Repositório com fim acadêmico para as aulas de Séries Temporais.

```
series-temporais/
├── atividades/   # Atividades desenvolvidas em aula

└── scripts/      # Scripts Python utilitários e de apoio
```

Cada pasta possui seu próprio `README.md` com instruções de organização interna.
