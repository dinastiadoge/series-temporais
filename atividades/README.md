# Atividades

Pasta destinada às atividades desenvolvidas durante as aulas de Séries Temporais.

## Estrutura sugerida

Organize cada atividade em uma subpasta numerada ou com nome descritivo:

```
atividades/
├── atividade-01/
│   ├── enunciado.pdf
│   └── resolucao.ipynb
├── atividade-02/
│   └── ...
└── ...
```
